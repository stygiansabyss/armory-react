Embed Options

Add options in the head of your webpage. Ensure it is set immediately and not after the page has loaded (don't use inside document.onReady or equivalent).

lang

  • 'en' (ENGLISH)
  • 'fr' (FRENCH)
  • 'de' (GERMAN)
  • 'es' (SPANISH)
  • 'zh' (CHINESE)
  • 'ru' (RUSSIAN [limited support])
document.GW2A_EMBED_OPTIONS = {
  lang: 'en',
};

Styling Embeds

To style any embed simply target the embed with the embed name, e.g:

.gw2a-character-embed
.gw2a-items-embed
.gw2a-skills-embed
.gw2a-tooltip-embed
.gw2a-specializations-embed

Character

  • data-armory-embed="character"
  • data-armory-name="{CHARACTER_NAME}"
<div
  data-armory-embed="character"
  data-armory-name="Blastrn"
>
</div>

Skills

  • data-armory-embed="skills"
  • data-armory-ids="{IDS_COMMA_SEPARATED}" | use id:-1 for blank skills
  • data-armory-size="{NUMBER}" (optional)
  • data-armory-blank-text="{STRING}" (optional, if missing, "optional" is used in its place)
<div
  data-armory-embed="skills"
  data-armory-ids="5507,5508,5510,5507"
>
</div>

Items

  • data-armory-embed="items"
  • data-armory-ids="{IDS_COMMA_SEPARATED}" | use id:-1 for blank items
  • data-armory-{ITEM_ID}-stat="{ITEMSTAT_ID}" (optional)
  • data-armory-{ITEM_ID}-skin="{SKIN_ID}" (optional) | Sets the item icon as the skin
  • data-armory-size="{NUMBER}" (optional)
  • data-armory-blank-text="{STRING}" (optional, if missing, "optional" is used in its place)
<div
  data-armory-embed="items"
  data-armory-ids="7139,8633,46774,74847,993,332211,109"
>
</div>

>
</div>
